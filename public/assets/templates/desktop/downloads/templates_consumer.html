The consumer template consists of the following fields:


Column Name Column Description
Number Serial or row number
Scheme The name of the Scheme from the Scheme Import Template
Habitation The name of the Habitation from the Habitation Import Scheme
Account Number Account number
New Account New account for departmental record keeping
Old Account Old account for departmental record keeping
Name Name of consumer
Fathers Name Father's name of consumer
Address Consumers Address
Disconnection Status Consumer Account Status: 0=Connected, 1=Disconnected
Connection Date Date when the account was connected
Disconnection Date Date when the account was disconnected
Economic Status Economic Status of consumer: APL, BPL
Category Consumer category: Gen, SC, BC
Prepay Months remaining The number of months the consumer as already paid in advance
Balance Consumer's account balance
Create Connection True=Create Connection Entry, False=Skip
Connection Fee If Create Connection=true (as defined above), the amount to use for this conncetion fee entry.
Security Amount of Security Deposit already collected. This amount is subtracted from the Consumers balance as specified above
Create Balance Correction True=Create balance correction entry, False=Skip
Balance Date The date to use for the balance correction entry.
