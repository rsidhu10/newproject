Downloads: Habitation Template



Download CSV  Download XLS 


The habitation template consists of the following fields:


Column Name Column Description
Number Serial or row number
Wing The name of the wing
Circle The name of the Circle
District The name of the District
Division The name of the Division
Subdivision The name of the Subdivision
Block The name of the Block
Panchayat The name of the Panchayat
Village The name of the Village
Habitation The name of the Habitation
Scheme The name of the Scheme, must match the name from Scheme import template
Connections Number of connections in this habitation
Type Habitation type: MH, OH
Status Connection status of the habitation: FC, PC, NC
P LPCD Present LPCD capacity of the habitation
Latitude Latitude of the habitation in decimal format
longitude Longitude of the habitation in decimal format
MIS ID MIS ID of the habitation
Hadbast_no Hadbast number for the habitation
Gen Pop General population of habitation
SC Pop Scheduled caste population of habitation
Gen HH Number of general households in habitation
sc_hh Number of scheduled caste households in habitation
Late Fee Type Late fee type for the habitation: 0=Disabled, 1=Percentable of Bill, 2=Fixed Amount
Late Fee Percent Percentage of Late Fee
Late Fee Fixed Amount of late fee if paid in the month dye
Late Fee Fixed Month Amount of late fee if paid after month due
Enable New Connection Fee True=Enable new connection fee, False=Disable
New Connection Fee GC New connection fee for General category
New Connection Fee BC New connection fee for BC Category
New Connection Fee SC New connection fee for SC Category
Enable Disconnection Fee True=Enable disconnection fee, False=Disable
Disconnection Fee GC Disconnection fee for General category
Disconnection Fee BC Disconnection fee for BC category
Disconnection Fee SC Disconnection fee for SC category
Enable Reconnection Fee True=Enable reconnection fee, False=Disable
Reconnection Fee GC Reconnection fee for General category
Reconnection Fee BC Reconnection fee for BC category
Reconnection Fee SC Reconnection fee for SC category
Deposit Amount of deposit required for consumers
Short Amount If the balance is short of this amount, the consumers bill will still be closed
Prepay Enabled True=Enable prepay, False=Disable
Prepay Months Minimum number of months that must be paid in advance
Prepay Amount Amount charged for the Prepay Months defined above
